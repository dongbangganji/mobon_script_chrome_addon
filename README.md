## 설치 방법
1. 크롬 창을 연 뒤 url : chrome://extensions/ 입력
2. 개발자 모드 활성화
   ![img.png](img/img.png)
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 소스 디렉토리 폴더선택
   ![img.png](img/img2.png)
5. mobon script check 가 설치 된게 확인 되면 완료
6. git pull 받고는 항상 아래 이미지에   
   새로고침 버튼을 클릭 해서 버전 확인 필수!
